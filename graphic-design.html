Graphic Design Course

Duration: 4 months

Course Overview

This Graphic Design course will introduce you to the world of visual design. You will learn how to create stunning graphics using tools like Adobe Photoshop and Illustrator. The course will cover typography, color theory, branding, and web design principles.

Course Curriculum

  • Introduction to Design Tools - Photoshop & Illustrator basics
  • Typography - Understanding fonts and layout
  • Color Theory - Effective color usage in design
  • Logo Design - Creating iconic logos and branding
  • Web Design Principles - Designing websites and apps
